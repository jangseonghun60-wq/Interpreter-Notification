Interpreter Notification — Documentation

This guide explains how to add, use, and remove the Interpreter Notification Zoom App.

How to Add the App

  1. Go to the Zoom App Marketplace and search for "Interpreter Notification".
  2. Click Add to install the app to your Zoom account.
  3. Authorize the app when prompted.

How to Use the App

  1. Start or join a Zoom meeting.
  2. Open the Interpreter Notification app from the Apps panel in the Zoom meeting client.
  3. At the start of the meeting, click "Send Invitation to All Participants" so participants can receive notifications.
  4. When needed, click one of the preset message buttons to send a notification banner to all participants:
    • Please speak closer to the mic
    • Please speak slower
    • Audio is unclear
  5. The notification banner will appear on participants' screens for 5 seconds and disappear automatically.
Note: Only the interpreter needs to install this app. Participants do not need to install anything.

How to Remove the App

  1. Go to marketplace.zoom.us.
  2. Click on your profile and select Manage.
  3. Find "Interpreter Notification" and click Remove.
